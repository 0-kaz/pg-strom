周辺ツール

本章ではPG-Stromの提供する周辺コマンドについて説明します。

gpuinfo

概要

gpuinfoはシステムに搭載されているGPUの情報を収集し、標準出力に出力するためのツールです。

PostgreSQLシステムの起動時、PG-Stromも内部的にgpuinfoコマンドを呼び出してGPUの情報を収集しています。

postmasterプロセスでCUDAドライバを初期化するcuInit()APIを呼び出した後、ユーザからの接続を受け付けるためにfork(2)システムコールを呼び出すと、子プロセス側でCUDAドライバAPIを使用できない状態に陥ってしまうため、GPUの情報取得のみを別プロセスに切り分けるために、このような構造を取っています。

使い方

gpuinfoコマンドをオプションなしで実行すると、GPUのデバイス属性のうち重要なものだけを人間に読みやすい形式で出力します。

その他のオプションは以下の通りです。

  • -d : detailed output
    • その環境でサポートされている全てのGPUデバイス属性を出力します。
  • -m : machine readable format
    • プログラムでパースするのに適した形式で出力します。
  • -h : help message
    • gpuinfoのコマンドラインオプションを出力して終了します。
$ gpuinfo -h
usage: gpuinfo [-d][-m][-h]
  -d : detailed output
  -m : machine readable format
  -h : shows this message
